# odin-recipes
A fake recipe site, made to learn web development from [The Odin Project](https://www.theodinproject.com).

## How to use
To visit the website, go to [natterjackdaw.github.io/odin-recipes](natterjackdaw.github.io/odin-recipes).

I would not advise you to follow the recipes, they are just for practicing web development.

Images are not my own, each recipe page contains a link to the recipe that they came from.
